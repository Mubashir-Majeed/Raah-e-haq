## 📋 **4. POSTMAN SETUP (CORRECT WAY)**

### **For Registration (Use form-data):**
1. **Method:** POST
2. **URL:** `http://127.0.0.1:9000/api/auth/register`
3. **Body:** Select `form-data`
4. **Add all required fields** as KEY-VALUE pairs
5. **Add file uploads** as FILE type
6. **Send request**

### **For Login (Use raw JSON):**
1. **Method:** POST
2. **URL:** `http://127.0.0.1:9000/api/auth/login`
3. **Body:** Select `raw`
4. **Content-Type:** Select `JSON`
5. **Add JSON data**
6. **Send request**
